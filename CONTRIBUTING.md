# Team workflow / Правила работы команды

## Before starting work / Перед началом работы
1. git pull

## After finishing changes / После завершения работы
1. git add .
2. git commit -m "описание изменений"
3. git push

## Rules / Правила
- Do not overwrite other sections of the page.
- Each member works on their own section.

- Не изменяйте секции, за которые отвечает другой участник.
- Каждый участник работает над своей частью страницы.
